# ⚡️ koex-boilerplate
Build and deploy Nodejs applications very quickly

### Stack 📦️
- koa.js
- knex
- pm2
- Docker

### Usage 🔧
- yarn install (or npm install)
- yarn run local

### Deploy(flyio) 🚀
- [Sign in flyio](https://fly.io/app/sign-in)
- brew install flyctl
- flyctl auth login
- flyctl launch
- flyctl deploy
